#  Projekt konsept

Planen er å legge spillprosjektet mitt in i nettsiden. Det skal være en startside med en åpen chatt for alle på nettsiden, den skal brukes til å snakke om spillet eller andre ting. Det andre elementet på startsiden er en knapp med en href link/tag som tar deg til siden med spillet. 

# Vertøy jeg skal bruke
Jeg skal bruke HTML og CSS for å lage selve nettsiden. Jeg skal også sette opp database med mariadb på pien gjennom terminalen med ssh. Jeg trenger også logikk språk (python) fordi jeg vil ha innlogging og litt annet, da må koden vite når den skal legge en ny bruker inn i tabellen.
